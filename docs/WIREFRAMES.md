# Quillty - Wireframes

This document contains low-fidelity wireframes for key screens in the Quillty application.

**Design Principles:**
- Mobile-first (tablet-optimized for quilting at sewing table)
- Large touch targets (44x44px minimum) for older demographics
- High contrast, readable typography (16px minimum)
- Masonry grid for visual content
- Clear visual distinction between blocks and patterns

## Design Specifications

### Color Palette (Suggested)

| Color | Hex | Usage |
|-------|-----|-------|
| Primary | #2563EB | Buttons, links, accents |
| Secondary | #7C3AED | Partner badges, premium |
| Success | #10B981 | Success states, free badge |
| Warning | #F59E0B | Warnings, pending |
| Error | #EF4444 | Errors, destructive |
| Background | #FFFFFF | Main background |
| Surface | #F9FAFB | Cards, elevated surfaces |
| Text Primary | #111827 | Headings, body text |
| Text Secondary | #6B7280 | Captions, metadata |
| Border | #E5E7EB | Dividers, card borders |

### Typography (Suggested)

| Element | Size | Weight |
|---------|------|--------|
| H1 | 28px | Bold |
| H2 | 24px | Semibold |
| H3 | 20px | Semibold |
| Body | 16px | Regular |
| Caption | 14px | Regular |
| Small | 12px | Regular |

### Spacing

- Base unit: 4px
- Common values: 8px, 12px, 16px, 24px, 32px, 48px

### Touch Targets

- Minimum: 44x44px
- Recommended: 48x48px for primary actions
